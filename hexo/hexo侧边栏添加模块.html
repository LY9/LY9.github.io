Hexo侧边栏添加模块

发表于 2017-09-01 | 更新于 2018-08-20 | 分类于 Hexo |
字数统计 | 阅读时长

在侧边栏添加了一个书籍分享的模块,记录如下.

环境:hexo+next主题

新建page

执行如下命令,source目录下产生book目录和index.md文件

hexo new page "book"

编辑index.md

写什么都随意

编辑主题配置文件

打开themes/next/_config.yml文件编辑.

menu:
    home: /
    archives: /archives
    tags: /tags
    categories: /categories
    book: /book    #添加book

menu_icons:
    enable: true
    home: home
    about: user
    categories: th
    book: book

添加语言

打开themes\next\languages 编辑相应的语言.

menu:
    home: 首页
    archives: 归档
    categories: 分类
    tags: 标签
    book: 书籍分享

查看效果

在hexo根目录执行如下命令

hexo s --debug

进入 http://localhost:4000/ 查看
