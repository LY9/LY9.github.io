Android开发中使用到的ADB命令

发表于 2018-09-13 | 更新于 2018-09-13 | 分类于 Android |
字数统计 | 阅读时长

抓取Crash日志

adb logcat -s AndroidRuntime

打印应用内存

adb shell dumpsys meminfo <package-name>

Debug包导出tarce文件，分析OOM

adb shell ps
adb shell am dumpheap <PID> /data/local/tmp/<file-name>.hprof

打印Activity启动时长

adb shell am start -W -n <package-name>/<package-name>.<activity-name>
例如：adb shell am start -W -n com.hongxiu.app/com.hongxiu.app.activity.SplashActivity

打印通知栏

adb shell dumpsys notification

打印启动intent

adb logcat -v time -s ActivityManager

打印系统事件的日志

adb logcat -v threadtime -b events

无线连接

adb tcpip <端口号>
断开手机等设备
adb connect <设备ip地址>
查看结果
adb devices
断开连接
adb disconnect <设备ip地址>
失败从adb kill-server开始重试

查看前台Activity

adb shell dumpsys activity activities | grep mFocusedActivity

查看手机上应用列表

adb shell pm list packages
查看应用安装目录
adb shell pm list packages -f
查看已卸载的应用
adb shell pm list packages -u
查看系统应用
adb shell pm list packages -s

获取设备型号

adb shell getprop ro.product.model

查看Android Id

adb shell settings get secure android_id

获取Android版本

adb shell getprop ro.build.version.release

状态栏和导航栏显示隐藏

adb shell settings put global policy_control <key-values>

<key-values> 可由如下几种键及其对应的值组成，格式为 <key1>=<value1>:<key2>=<value2>。

key 含义
immersive.full 同时隐藏
immersive.status 隐藏状态栏
immersive.navigation 隐藏导航栏
immersive.preconfirms ?

这些键对应的值可则如下值用逗号组合：

value 含义
apps 所有应用
* 所有界面
packagename 指定应用
-packagename 排除指定应用

例如adb shell settings put global policy_control immersive.full=*，表示设置在所有界面下都同时隐藏状态栏和导航栏
